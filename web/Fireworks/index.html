Interactive Firework Shooter

Controls

Controls

  • Press the 'A' and 'D' keys to move the ship!
  • Tap 'space' to shoot fireworks!
  • Or click the shoot button!
  • Change the size of firework
  • Choose firework in dropdown menu

About

This mini web application is based off of the math behind phyllotaxis, or the arrangement of lateral organs of plants. It uses the phyllotaxis equation to create the fireworks by calling a function at a set frame rate. Phyllotaxis is a very interesting subject and you can read more here.
